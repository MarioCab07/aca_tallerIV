# :material-brain: Fundamentos de la IA

La IA se basa en algoritmos capaces de identificar patrones y aprender a partir de datos.

## Tipos principales

### Machine Learning

Permite que los sistemas aprendan sin ser programados explícitamente para cada tarea.

### Deep Learning

Utiliza redes neuronales artificiales inspiradas en el cerebro humano.

!!! tip "Dato importante"
    Deep Learning es la tecnología detrás de sistemas como ChatGPT, reconocimiento facial y vehículos autónomos.

## Ejemplo de entrenamiento

```python
datos = [1, 2, 3, 4, 5]

for dato in datos:
    print("Procesando:", dato)
```
