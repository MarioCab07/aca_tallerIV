# :material-rocket-launch: Aplicaciones de la IA

La IA está presente en numerosos ámbitos de la sociedad moderna.

## Ejemplos de uso

=== "Salud"

    ```python
    diagnostico = {
        "paciente": "Juan",
        "riesgo": "Bajo"
    }
    ```

=== "Educación"

    ```python
    estudiante = {
        "nombre": "Ana",
        "progreso": 92
    }
    ```

=== "Ciberseguridad"

    ```python
    amenaza_detectada = True

    if amenaza_detectada:

    ```

| Área      | Beneficio                     |
